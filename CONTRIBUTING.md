## Development Notes

This project was developed as an individual academic assignment worth 0–4 bonus points.
The game architecture, evidence scenario, hallucination taxonomy, and pedagogical design
are original work. The AI integration uses Google Gemini 2.5 Flash via the
`@google/generative-ai` SDK.

## Running the Project

See `README.md` for full setup instructions. Quick start:

```bash
npm install
cp .env.example .env   # edit .env if you want Live Gemini mode
npm run dev
```
