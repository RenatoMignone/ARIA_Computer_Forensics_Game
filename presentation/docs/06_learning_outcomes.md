# 06 Learning Outcomes

## Main Goal

ARIA trains evidence-based judgment in AI-assisted forensic analysis.

After playing, students should understand that AI output can support an investigation but cannot replace verification.

## Forensic Skills

Students practice:

- Reading email authentication fields such as SPF, DKIM, Return-Path, and originating IP.
- Inspecting media metadata such as creation time, encoder, sample rate, bitrate anomalies, and compression artifacts.
- Reviewing PDF metadata, payment details, encoded creator fields, and signature status.
- Reading network source and destination activity.
- Checking hash values and chain-of-custody indicators.
- Comparing timestamps across artifacts.
- Building a simple incident timeline.
- Connecting related evidence.
- Producing a final report that can be explained.

## AI Literacy Skills

Students practice:

- Detecting unsupported technical claims.
- Recognizing confident but unverified wording.
- Separating direct evidence from inference and speculation.
- Rejecting hallucinations without rejecting every AI suggestion.
- Calibrating confidence based on available support.

## Cybersecurity Concepts

The scenario introduces:

- Spear phishing.
- Email spoofing.
- Business email compromise.
- Voice cloning.
- Deepfake indicators.
- Suspicious PDF generation.
- Tor-related activity.
- Possible exfiltration.
- Social engineering pressure.

## Assessment Behaviors

The game rewards:

- Correctly verifying true claims.
- Correctly identifying hallucinated claims.
- Finding cross-evidence links.
- Using terminal commands to inspect and validate evidence.
- Submitting the final report after the case claim set is complete.

The game penalizes:

- Accepting hallucinations as true.
- Rejecting true claims without support.
- Attempting prompt injection or suspicious ARIA manipulation.
- Using hints in modes where hints are available.

## Expected Outcome

A successful player should be able to explain:

- Which claims were supported by evidence.
- Which claims were hallucinated.
- Which artifacts were used to decide.
- How the evidence connects across the case.
- Why the final conclusion is defensible.

## Professional Mindset

The intended habit is simple: treat AI as an assistant, not an authority. Verify first, conclude later.
