# Documentation

This folder contains operational documentation for running, assessing, and deploying ARIA.

- [`setup-and-api-key.md`](setup-and-api-key.md): local setup, production build, `.env` creation, and optional Gemini API key configuration.
- [`gameplay-and-assessment.md`](gameplay-and-assessment.md): complete player workflow, assessment gates, scoring behavior, and final report conditions.
- [`deployment.md`](deployment.md): GitHub Pages deployment flow and public demo notes.

Project context and design-reference material remains in `presentation/` so operational documentation and broader rationale stay clearly separated.
